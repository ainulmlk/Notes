Passive Voice

Object + BE (conjugated) + Past Participle

Object + BE + past participle (+ by + agent)
is / am / are + past participle
Active
They build houses.
Passive
Houses are built.
Example
The road is repaired every year.
was / were + past participle
Active
They built the bridge.
Passive
The bridge was built.
Teacher example
He was recruited two weeks ago.
has / have + been + past participle
Active
They have completed the project.
Passive
The project has been completed.
Teacher example
This document has not been understood by the staff.
will + be + past participle
Active
They will launch the product.
Passive
The product will be launched.
Another
The meeting will be postponed.
modal + be + past participle
Example
The work must be finished.
Example
The problem can be solved.
Example
Rules must be followed.
⚠️ Hold up = delayed (passive)
The train was delayed. / The train has been held up.

Third Conditional

For impossible / imaginary past situations. It didn't happen — but what if it had?

IF + had + past participle → would have + past participle
Teacher Example
None of this would have happened if I hadn't been fired from Apple.
More Examples
If I had studied harder, I would have passed.
If we had left earlier, we would have caught the train.
If she had called me, I would have come.
Inverted Form (Advanced)

Drop "if" and invert the subject/verb.

Normal
If I had known…
Inverted
Had I known…
Had I known, I would have helped.
⚠️ had better
✅ Correct
Students had better start working.
❌ Wrong
Students had better to start working.

had better + BASE VERB (no "to")
