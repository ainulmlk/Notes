Passive Voice

Object + BE (conjugated) + Past Participle

Object + BE + past participle (+ by + agent)
⭐ When to use the Passive — this is the key
ReasonExample
Agent is unknownMy car was stolen. (we don't know who)
Agent is obviousHe was arrested. (obviously by police)
Agent is unimportantThe report has been sent. (who sent it doesn't matter)
Formal / professional writingThe system will be upgraded next week.
Focus on the action, not the personThree people were injured. (not "a car hit three people")
Scientific / technical writingThe samples were tested at 37°C.
💡 Rule of thumb: use passive when the object matters more than the subject.
Active: Apple launched the iPhone. → focus on Apple
Passive: The iPhone was launched in 2007. → focus on the iPhone

Active

Use to express obligation, possibility, or advice without naming who does it.

When to add "by + agent" — and when not to

✅ Include "by" when the agent adds useful info

This play was written by Shakespeare.
The photo was taken by a professional.

❌ Drop "by" when it's obvious or irrelevant

He was arrested. (not "by police")
The report was sent. (we don't care who)
⚠️ Teacher's passive examples

Conditionals

5 types — zero to mixed. Your teacher focuses on 3rd but all can appear.

Quick Reference — All Conditionals
TypeIF clauseMain clauseMeaning
Zeropresent simplepresent simpleAlways true / scientific fact
1stpresent simplewill + infinitiveReal / likely future
2ndpast simplewould + infinitiveImaginary present/future
3rdpast perfectwould have + ppImaginary past (impossible)
Mixedpast perfectwould + infinitivePast cause → present result
IF + present simple → present simple

Use for facts, scientific truths, and things that are always the result of something.

If you heat water to 100°C, it boils.
If it rains, the ground gets wet.
If I drink coffee late, I can't sleep.
💡 You can also replace "if" with "when" — same meaning: When water reaches 100°C, it boils.
IF + present simple → will + infinitive

Use when something is possible and you believe it could actually happen.

If it rains tomorrow, I will stay home.
If you study hard, you will pass.
If she calls, I will tell her the news.
⚠️ Never use "will" in the IF clause:
❌ If it will rain… → ✅ If it rains…
IF + past simple → would + infinitive

Use for situations that are unlikely, hypothetical, or contrary to the present reality.

If I had more time, I would learn Japanese.
If she lived in Paris, she would visit the Eiffel Tower every day.
If I were you, I would apply for the job.
💡 After "if", use were for all persons (formal):
If I were rich… / If she were here… (not "was" in formal writing)
IF + had + past participle → would have + past participle

Use for imaginary past situations — it didn't happen, but what if it had?

Teacher
None of this would have happened if I hadn't been fired from Apple.
If I had studied harder, I would have passed.
If we had left earlier, we would have caught the train.
If she had called me, I would have come.

Inverted form (drop "if", invert subject/verb):

Normal
If I had known…
Inverted
Had I known…
Had she arrived earlier, she would have met him.
IF + had + past participle → would + infinitive (now)

Use when a past event affects the present situation. The past didn't happen, so the present is different.

If I had studied medicine, I would be a doctor now.
If she hadn't missed the flight, she would be here now.
If he had saved his money, he wouldn't need a loan today.
💡 How to tell 3rd vs Mixed:
3rd: Both halves are about the past → would have + pp
Mixed: Past IF, present result → would + infinitive (no "have")
⚠️ had better — not a conditional but often confused
✅ Correct
Students had better start working.
❌ Wrong
Students had better to start working.

had better + BASE VERB (no "to", no -ing). Meaning: strong advice / warning.

Gerund vs Infinitive

-ING or TO + verb? The choice sometimes changes the meaning entirely.

⭐ Verbs that CHANGE MEANING — classic exam trap

REMEMBER

remember + ING
I remember locking the door. → I locked it and I recall doing it (past memory)
remember + TO
Remember to lock the door. → Don't forget to do it (future task)

FORGET

forget + ING
I'll never forget meeting her. → I met her and I won't forget that moment
forget + TO
I forgot to call him. → I didn't call him (future task not done)

STOP

stop + ING
She stopped smoking. → She no longer smokes (quit the habit)
stop + TO
She stopped to smoke. → She paused in order to have a cigarette

TRY

try + ING
Try adding more salt. → Experiment / see if it helps
try + TO
I tried to open the door. → I made an effort (may or may not succeed)

REGRET

regret + ING
I regret saying that. → I'm sorry I said it (past action)
regret + TO
I regret to inform you… → Formal: I'm sorry to tell you this now
